Multi-Campus Points MVP

多校区学生积分管理系统

当前版本已升级为 Supabase MVP。老师页、榜单页、学生主档页、班级管理页和管理配置页都已接入真实数据库，并新增最小登录与角色体系。

打开老师加分页 打开大屏展示页 打开管理配置页 打开学生管理页 打开班级管理页
T

teacher.html

响应式老师控制台，支持班级创建、学生加入和真实积分流水写入。

 D

display.html

大屏三榜单展示，真实读取总分榜与进步榜，徽章榜当前为占位数据。

 A

admin.html

管理配置页已接入 `point_rules` 和 `level_tiers`，用于维护积分规则和段位阈值。

 S

students.html

学生管理页，支持学生主档列表、手动新增、CSV 导入预览和疑似重复提醒。

 C

classes.html

班级管理页，支持班级列表、新建班级、查看详情和从学生主档搜索加入班级。

使用说明

请先按 README 在 Supabase 中执行 SQL，再运行 `npm run dev` 打开本地地址进行演示。
