Multi-Campus Points MVP

多校区学生积分管理系统

当前版本已升级为 Supabase MVP。老师页、榜单页、学生主档页、班级管理页和管理配置页都已接入真实数据库，并新增最小登录与角色体系。

进入管理台 学生主档 班级管理 老师工作台 积分大屏
A

admin.html

配置积分规则、段位阈值、徽章规则与老师账号，作为试运行期间的统一运营后台。

 S

students.html

维护学生主档，支持搜索、手动新增、CSV 导入预览、查重提示与状态展示。

 C

classes.html

查看班级列表、筛选班级、创建新班、打开详情并从学生主档中检索成员加入班级。

 T

teacher.html

老师端支持切校区、切班级、单人加分、整班加分、快速建班、补录与兑换等核心动作。

 D

display.html

积分大屏展示总分榜、进步榜和徽章榜，适合前台电视或公开展示场景。

试运行说明

当前首页仅对已登录管理员开放；老师账号会自动跳转到老师工作台。页面通过本地 HTTP 服务打开，禁止直接双击 HTML 文件进入。
